QuoteBot

A conversational chatbot that recommends quotes based on a user's mood or request. The system uses Natural Language Processing (NLP) to understand user intent and deliver relevant quotes such as motivational, love, or humorous quotes.

## Project Overview

The Quotes Recommendation Chatbot using Rasa NLU is a conversational AI system that provides personalized quotes based on a user's mood, interests, or requests. It helps users quickly find motivation, inspiration, humor, or emotional support through natural language interaction.

Built using Rasa NLU, the chatbot understands user intent and recommends quotes from categories such as Motivation, Inspiration, Success, Love, and Humor. It enhances user engagement by delivering context-aware responses and meaningful conversations.

The system also supports web integration through a REST API, allowing users to interact with the chatbot via a browser-based interface connected to the Rasa backend.

Features

1) NLP-based intent detection

2) Quote recommendations based on mood or request

3) Multiple quote categories:

    Motivational

    Love

    Humor

4) Conversational dialogue handling using Rasa

5) Custom actions for dynamic responses

6) Simple frontend interface for user interaction


Technologies Used:

1) Python

2) Rasa Framework

3) Natural Language Processing (NLP)

4) HTML / CSS / JavaScript

5) Machine Learning
